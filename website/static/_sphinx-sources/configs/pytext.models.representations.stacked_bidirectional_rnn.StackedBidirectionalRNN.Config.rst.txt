  Configuration class for `StackedBidirectionalRNN`.
  
  .. attribute:: hidden_size
  
     Number of features in the hidden state of the RNN.
     Defaults to 32.
  
     :type: int
  
  .. attribute:: num_layers
  
     Number of recurrent layers. Eg. setting `num_layers=2`
     would mean stacking two RNNs together to form a stacked RNN,
     with the second RNN taking in the outputs of the first RNN and
     computing the final result. Defaults to 1.
  
     :type: int
  
  .. attribute:: dropout
  
     Dropout probability to use. Defaults to 0.4.
  
     :type: float
  
  .. attribute:: bidirectional
  
     If `True`, becomes a bidirectional RNN. Defaults
     to `True`.
  
     :type: bool
  
  .. attribute:: rnn_type
  
     Which RNN type to use. Options: "rnn", "lstm", "gru".
  
     :type: str
  
  .. attribute:: concat_layers
  
     Whether to concatenate the outputs of each layer
     of stacked RNN.
  
     :type: bool
