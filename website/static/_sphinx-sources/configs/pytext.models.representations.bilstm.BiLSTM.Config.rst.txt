  Configuration class for `BiLSTM`.
  
  .. attribute:: dropout
  
     Dropout probability to use. Defaults to 0.4.
  
     :type: float
  
  .. attribute:: lstm_dim
  
     Number of features in the hidden state of the LSTM.
     Defaults to 32.
  
     :type: int
  
  .. attribute:: num_layers
  
     Number of recurrent layers. Eg. setting `num_layers=2`
     would mean stacking two LSTMs together to form a stacked LSTM,
     with the second LSTM taking in the outputs of the first LSTM and
     computing the final result. Defaults to 1.
  
     :type: int
  
  .. attribute:: bidirectional
  
     If `True`, becomes a bidirectional LSTM. Defaults
     to `True`.
  
     :type: bool
